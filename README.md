# Python to R Wrapper

### Installation

1. You need to have R (3.4.2 or above) installed (https://www.r-project.org)

  ex: `brew install R`    

2. Install python 3.6+

### How to run?
This application is built on flask. The following command will start the app:

```
export FLASK_APP=main.py
flask run

```
